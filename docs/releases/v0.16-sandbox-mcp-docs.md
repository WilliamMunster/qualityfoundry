# Release Notes - v0.16-sandbox-mcp-docs

## 关键能力
- **L3 Container Sandbox**: 支持 `run_pytest` 容器化执行，具备禁网、只读挂载、资源限制与超时强杀能力。
- **L4 MCP Write Security**: 实现 `run_pytest` 受控写工具，集成认证→权限→策略→沙箱完整安全链。
- **UUID Runs 主路径**: 统一编排出口，支持启动、详情查看、证据下载与审计溯源。
- **文档中文化**: 全面汉化 `progress_baseline.md`，新增术语表对齐。

## 安全边界
- **Allowlist 约束**: MCP 写操作要求 `policy.tools.allowlist` 非空且包含目标工具。
- **Sandbox 强制**: MCP 写操作强制要求 `sandbox.enabled: true`。
- **隔离一致性**: `sandbox.mode: container` 不可用时将拒绝执行并记入审计，不进行静默降级。

## 验收结果
- **Release Gate**: `pytest -m smoke_fast` + `unit tests` + `ruff check` 全部通过。
- **Playwright 说明**: 部分 Playwright E2E 测试因环境缺失导致 skip / 非门禁。
- **Security Check**: 25 项 MCP 安全专项测试全部通过。
